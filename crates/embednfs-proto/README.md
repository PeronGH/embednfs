# embednfs-proto

[![crates.io](https://img.shields.io/crates/v/embednfs-proto)](https://crates.io/crates/embednfs-proto)

NFSv4.1 protocol types and XDR encoding/decoding for Rust.

This crate provides the low-level wire types used by [embednfs](https://crates.io/crates/embednfs). See the [main project README](https://github.com/PeronGH/embednfs) for full documentation.
